install node and npm

install angular cli:
  > sudo npm i -g @angular/cli

create project : 

	> ng new my-first-app
	? Would you like to add Angular routing? Yes
	? Which stylesheet format would you like to use? SCSS


quickly run the project:

from within the project, type:
  > ng serve -o


angular-routing:

   generate components:

      > mg generate component home
      or
      > ng g c home
	CREATE src/app/home/home.component.scss (0 bytes)
	CREATE src/app/home/home.component.html (19 bytes)
	CREATE src/app/home/home.component.spec.ts (614 bytes)
	CREATE src/app/home/home.component.ts (262 bytes)
	UPDATE src/app/app.module.ts (467 bytes)


generate services:
> ng generate service http
or
> ng g s http
CREATE src/app/http.service.spec.ts (323 bytes)
CREATE src/app/http.service.ts (133 bytes)



build production distribution of your app:

   > ng build --prod


then look under dist folder of your project


install lite-server via npm to test prod dist version.

   > npm i -g lite-server
